INSTANCE SEGMENTATION USING MASK R-CNN ON CUSTOM DATASET USING PYTHON

Used Detectron2's Mask R-CNN baseline package (pre-trained algorithm) and performed transfer learning on a custom dataset of 57 goat images. The goat images were first split into training and validation datasets and then annotated individually using the via annotation tool. A user-interface to showcase the project was built in Python (Flask) on Google colab with NGROK. If the zoom is not visible enough, the link to this project can be found using this link Instance Segmentation Colab Notebook. This link will give you access to the Flask application using NGROK Flask Web Application.

STATISTICAL ANALYSIS OF PROTEIN EXPRESSION FROM LAB SUBJECT BRAIN USING R AND RSTUDIO

In this project I computed the mean value of protein expression for the mice with injured brain and normal control mice, respectively, for each protein, computed the standard deviation of protein expression for the mice with injured brain and normal control mice, respectively, for each protein, performed a two-sample t-test to compare the mean protein expression in the mice with injured brain and normal brain control mice for each protein, sorted the protein according to the p-value from the t-tests in ascending order, reported the protein with significant mean expression between the mice with injured brain and the normal control mice with p-value < 0.05 and p-value < 0.01, respectively, and extracted the t value from the t-test and generated a plot of p-value vs. t value. This project can be found on my Github account using this link Mice Brain Data Analysis
