INSTANCE SEGMENTATION USING MASK R-CNN ON CUSTOM DATASET USING PYTHON

Used Detectron2's Mask R-CNN baseline package (pre-trained algorithm) and performed transfer learning on a custom dataset of 57 goat images. The goat images were first split into training and validation datasets and then annotated individually using the via annotation tool. A user-interface to showcase the project was built in Python (Flask) on Google colab with NGROK. I did not pay much attention to the styling on the web app for this project. The main component of interest for the web app was the backend and the model. The goal is to show the user that the web app has functionality and that my model works. The link to this project can be found using this link Instance Segmentation Colab Notebook. This link will give you access to the Flask web application using an NGROK token Flask Web Application.

STATISTICAL ANALYSIS OF PROTEIN EXPRESSION FROM LAB SUBJECT BRAIN USING R AND RSTUDIO

In this project I computed the mean value of protein expression for the mice with injured brain and normal control mice, respectively, for each protein, computed the standard deviation of protein expression for the mice with injured brain and normal control mice, respectively, for each protein, performed a two-sample t-test to compare the mean protein expression in the mice with injured brain and normal brain control mice for each protein, sorted the protein according to the p-value from the t-tests in ascending order, reported the protein with significant mean expression between the mice with injured brain and the normal control mice with p-value < 0.05 and p-value < 0.01, respectively, and extracted the t value from the t-test and generated a plot of p-value vs. t value. This project can be found on my Github account using this link Mice Brain Data Analysis

DATA PREPARATION AND EXPLORATORY DATA ANALYSIS ON KAGGLE'S SPOTIFY DATASET

Data collection, data preparation, and data exploration/analysis are the most important components in a data science project. If these components are not accurate then the machine learning portion of the process can fail dramatically, resulting in one of the two pairs: high bias - low variance or low bias - high variance. High bias - low variance may lead to an under-fitting model while low bias - high variance may lead to an over-fitting model. The goal is to achieve a bias-variance tradeoff in the final model. In this project, I wanted to test out a few python packages that can be used in performing exploratory data analysis. I created a virtual environment (EDA2) to install the packages. I performed a few tasks in this project including checking for categorical variables, removing text data from a dataframe that may be used later, looping through missing data points, checking distributions of random variables, visualizing a change in data points over years for a random variable, checking for duplicate indices, checking for outliers in random variables, and normalizing a chosen random variable that was skewed. I used a series of transformations to see which one would give me the most symmetrical distribution (guassian or normal distribution). I purposefully did not use scikit-learn since the goal of this project was not to build a model or make any predictions. This project can be found on my github account using this link Exploratory Data Analysis Using Python and Jupyter Notebook
